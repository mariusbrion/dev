Outil Principal: Analyse de Trajets Domicile-Travail

Cet outil vous guide à travers un processus complet pour transformer vos listes d'adresses en une analyse cartographiée des trajets vélos potentiels.

Les étapes clés :

  1. Étape 1 : Préparation du Fichier
    Commencez avec un fichier d'adresses à 4 colonnes : Rue (salarié), Commune (salarié), Code postal (salarié), et Commune (site travail employeur). Plus ces adresses sont détaillées (ex: style Google Maps), plus le résultat sera précis, un outil de correction et développement des adresses par IA est fourni dans la section "plus d'outils...". Créer un fichier .csv est très simple depuis Excel il vous suffit de faire "enregistrer sous"et de choisir le format .csv
  2. Étape 2 : Transformation Automatique
    L'étape suivante consiste à transformer le fichier en simplement deux colonnes : une pour l'employé et l'autre pour l'employeur.
  3. Étape 3 : Géocodage (Adresses → Coordonnées)
    En récupérant le fichier et en allant à l'étape d'après, le site va "géocoder" vos adresses (les transformer en latitude et longitude) pour que le site puisse ensuite calculer les trajets.
  4. Étape 4 : Calcul des Itinéraires
    Une fois votre fichier téléchargé, vous pouvez le glisser dans l'étape suivante. Le site calculera les itinéraires vélos entre chaque point. (Vous pouvez rentrer votre propre clé API OpenRouteService pour être sûr de ne pas être bloqué sur le nombre de requêtes).
  5. Étape 5 : Téléchargement des Résultats
    Vous obtiendrez trois fichiers : un .csv qui vous permettra de connaître la part en % d'employés en fonction de la distance, et deux fichiers .geojson (formats utilisés pour la cartographie, voir le tutoriel).
